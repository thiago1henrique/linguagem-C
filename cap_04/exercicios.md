# Indique quais afirmações são FALSAS e quais são Verdadeiras:

<br/>

* 1.1 - A condição dentro de um laço while e do... while tem que ser colocada sempre dentro de parênteses. 
``` Verdadeiro ```

* 1.2 - Os laços while e for executam <b>SEMPRE</b>, pelo menos uma vez, o corpo do laço.
```Falso```

* 1.3 - O laço do... while executa SEMPRE, pelo menos uma vez, o corpo do laço
```Verdade```

* 1.4 - As três componentes, dentro do parênteses, do laço for são todas obrigatórias.
```Falso```

* 1.5 - As três componentes, dentro do parênteses, do laço for são todas facultativas.
```Verdadeiro```

* 1.6 - Quando um laço for se tem que realizar mais do que uma carga inicial ou mais do que um incremento, as diversas instruções, em cada uma das componentes, devem ser separadas por virgula e não ponto-e-virgula, de forma a manter o formato do laço for.
```Verdadeiro```

* 1.7 - A instrução de um laço é sempre executada.
```Falso```

* 1.8 - No laço for, o número de vezes que as cargas iniciais são executadas é sempre igual ao número de iterações do laço
```Falso```

* 1.9 - No laço for ou while, o número de vezes que a condição é testada é sempre igual ao número de iterações do laço.
```Falso```

* 1.10 - No laço for, o número de vezes que a instrução é excutada é sempre igual ao número de vezes que é executada a pós-instrução.
```Verdadeiro```

* 1.11 - Um laço, quando está dentro de outro laço, necessita de chaves.
```Falso```

* 1.12 - Não há limite para o número de laçosdentro de outros laços.
```Verdadeiro```

* 1.13 - No laço do...while, a instrução é sempre executa pelo menos uma vez.
```Verdadeiro```

* 1.14 - Em todos os laços, a condição é testada pelo menos uma vez.
```Verdadeiro```

* 1.15 - Os laços while e do...while incrementam automaticamente a variável de controle.
```Falso```

* 1.16 - Os laços são executados até que a condição escrita se verifique.
```Falso```

* 1.17 - O laço do...while aparece escrito menos frequentemente do que qualquer dos outros.
```Verdadeiro```

* 1.18 - Um break, quando presente dentro de um laço, termina o programa.
```Falso```

* 1.19 - No laço while, antes de se executar a instrução continue, deve-se atualizar a variável de controle do laço.
```Verdadeiro```

* 1.20 - Qualquer dos laços pode ser sempre escrito a partir do laço while
```Verdadeiro```
